# Acer Aspire V3-575 Upgrade Project

## Description
This project documents the complete upgrade of my Acer Aspire V3-575 laptop.
The goal was to replace the HDD with an SSD, expand RAM, and install Windows 11
to achieve a faster, more reliable, and optimized system.

## How to disassemble the computer
1. split all the screws
2. remove the side cover
3. Split the screws of the cover
4. take a butter knife, open the computer and do not turn it on and put the butter knife around

For a clearer tutorial, please watch this video: https://youtu.be/jFGtzY0USGM And to gather the computer, please check the video again

## How to install Windows
You need to install Windows only if you have changed your hard drive
It is necessary to have a USB key and a working computer

1. go to your browser
2. go to the official website to install Windows, here is the link
Windows 11: https://www.microsoft.com/en-ca/software-download/windows11
3. install the Windows 11 version support to install it on the key
4. open the downloaded application
5. follow the steps in the app (don't forget to choose the right key and connect it)
